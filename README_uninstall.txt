# Testimonials Manager
Testimonials Manager for Zen Cart 1.5.5f to 1.5.6 
Tested with php7.1 and php7.2 only
====================================
UNINSTALL:

To uninstall the contribution completely, follow these instructions.

Go to your admin.  Open tools>>Testimonials Manager

Click the remove button on the top and follow the instructions.

Manually delete all remaining files that came with this mod.
